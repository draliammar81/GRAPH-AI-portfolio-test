About the course

An 8‑week, live cohort program focused on no‑code AI tools for task automation and research productivity — with weekly workshops, help sessions, community support, feedback on assessments, and a capstone project showcase.

Module 1 — Websites, Documents & Presentations

How LLMs work; generate Markdown/LaTeX/HTML; build simple research sites; structure documents; AI presentation tools.

Module 2 — AI‑Automated Data Analysis

Use ChatGPT Code Interpreter; dashboards with TheBricks; process datasets with Julius; batch conversions; automated reports; spreadsheet flows.

Module 3 — GenAI for Research & Text Analysis

Efficient literature reviews (Elicit/LiteRev); text classification; sentiment analysis; knowledge maps; summarization pipelines; synthesis systems.

Module 4 — Custom GPTs & Chatbots

LLM chat architecture, context windows & RAG; custom GPTs; Botpress chatbots; knowledge base integration.

Module 5 — AI for Task Automation

Automation strategies; email response flows; research monitoring agents; browser control agents.

Module 6 — Communication & Outreach

Research summaries; figure/diagram design; video abstracts; prompts for visualization; content scheduling.

Module 7 — Capstone & Showcase

Peer collaboration, iterative coaching, and final public presentation of the project.

Bootcamp Features

Weekly live workshops • Live help sessions • Private forums • Personalized feedback • Capstone coaching • Certificate • Lifetime access to tutorials.

Grading (Overview)

Quizzes 15% • Assignments 35% • Attendance 10% • Final project 40% (Passing ≥ 80%; attend ≥ 5 workshops).
